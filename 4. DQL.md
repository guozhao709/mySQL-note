# 单表查询

会产生一个虚拟表

## select

1. 非表查询

```sql
SELECT VERSION();
```

2. 指定表

```sql
SELECT <列名>, <列名> FROM <表名>;
SELECT <表名>.<列名>, <表名>.<列名> FROM <表名>;
SELECT * FROM <表名>;
```

3. 查询并起别名

```sql
SELECT <列名> AS <别名>, <列名> AS <别名> FROM <表名>;
```

4. 去重

```sql
SELECT DISTINCT <列名> ... FROM <表名>;
```

5. 查询常数

```sql
SELECT  <列名> ..., 'GUOZHAO' AS master FROM <表名>;
```

相当于在虚拟表上最后再加一master列, 所有值为GUOZHAO, 当然可以用数据库的变量进行运算

空进行任何运算都为空

```sql
SELECT name, salary * IFNULL(pos, 0) AS ALLsalary FROM <表名>;
```

6. 显示表数据

```sql
DESC <表名>;
```

```

```

## 单行: 流程函数

```
IF(表达式, true_value, false_value)
IFNULL(列名, 替代值)  如果列名值为NULL, 则返回替代值, 否则返回列名值

CASE 列名 | 表达式
    WHEN 值1 THEN 结果1
    WHEN 值2 THEN 结果2
    ...
    ELSE 结果n
END

```

## 多行: 聚合函数

```
AVG(列名)  平均值=>数值类型
SUM(列名)  总和=>数值类型
COUNT(列名 / * / 1)  列名=>非NULL值的数量, * 所有行非空
MAX(列名)  最大值
MIN(列名)  最小值
```

多行函数对NULL是不关心的, 多行函数不能嵌套

# 分组查询

```
SELECT 列名, 聚合函数(列名) FROM 表名
[WHERE 条件]
GROUP BY 列名(分组依据) [HAVING 条件];
```

# 排序查询

```
SELECT 列名, 列名, 函数
FROM TABLE
ORDER BY 列名(排序依据) [ASC/DESC];
```

asc表示从小到大(正序), desc表示从大到小(倒序)

# 分割查询(mysql方言)

```
SELECT 列名, 列名, 函数
FROM TABLE
WHERE 条件
LIMIT 偏移量 行数
```

一般前端会传page和size, 根据page和size来确定分页

# 查询执行流程

关键字书写顺序

```
SELECT 列名 FROM 表名 
WHERE 筛选前置条件(不能写别名)
GROUP BY 分组字段 HAVING 分组后筛选条件
ORDER BY 排序字段(排序依据) [ASC/DESC];
LIMIT 偏移量 行数
```
