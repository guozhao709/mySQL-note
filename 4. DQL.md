# 单表查询

会产生一个虚拟表

## select
1. 非表查询
```sql
SELECT VERSION();
```

2. 指定表
```sql
SELECT <列名>, <列名> FROM <表名>;
SELECT <表名>.<列名>, <表名>.<列名> FROM <表名>;
SELECT * FROM <表名>;
```

3. 查询并起别名
```sql
SELECT <列名> AS <别名>, <列名> AS <别名> FROM <表名>;
```

4. 去重
```sql
SELECT DISTINCT <列名> ... FROM <表名>;
```
5. 查询常数
```sql
SELECT  <列名> ..., 'GUOZHAO' AS master FROM <表名>;
```
相当于在虚拟表上最后再加一master列, 所有值为GUOZHAO, 当然可以用数据库的变量进行运算

空进行任何运算都为空
```sql
SELECT name, salary * IFNULL(pos, 0) AS ALLsalary FROM <表名>;
```

6. 显示表数据
```sql
DESC <表名>;
```

# 运算符

## 算术运算符

```
+ - * /(浮点除) DIV(整数除) %
```
运算结果如果为错误, 则为NULL

## 比较运算符

```
>
>=
<
<=
=
<>  不等
IS NULL
IS NOT NULL
BETWEEN 值1 AND 值2     值是否在值1和值2之间(闭区间)
NOT BETWEEN 值1 AND 值2  值是否不在值1和值2之间(闭区间)
IN(值1, 值2, ...)   值是否在这一组值中
NOT IN(值1, 值2, ...)    值是否不在这一组值中
LIKE  (模糊查询, 通配符%, 表示匹配多个; _, 表示匹配一个)
NOT LIKE
```
正确返回1, 否则返回0

# 单行和多行函数

## 单行: 数值函数

```

SIGN(X)    返回符号(1, -1, 0)
PI()    返回圆周率
CEIL(X)     向上取整
FLOOR(X)    向下取整
LEAST(X1, X2, ...)  返回最小值
GREATEST(X1, X2, ...)  返回最大值
MOD(X, Y)  返回X除以Y的余数
RAND()  返回随机数
RAND(X)     相同的x返回相同的随机数
ROUND(X)    四舍五入, 整数

SQRT(X)     开平方根
```
