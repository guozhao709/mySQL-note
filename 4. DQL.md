# 单表查询

会产生一个虚拟表

## select
1. 非表查询
```sql
SELECT VERSION();
```

2. 指定表
```sql
SELECT <列名>, <列名> FROM <表名>;
SELECT <表名>.<列名>, <表名>.<列名> FROM <表名>;
SELECT * FROM <表名>;
```

3. 查询并起别名
```sql
SELECT <列名> AS <别名>, <列名> AS <别名> FROM <表名>;
```

4. 去重
```sql
SELECT DISTINCT <列名> ... FROM <表名>;
```
5. 查询常数
```sql
SELECT  <列名> ..., 'GUOZHAO' AS master FROM <表名>;
```
相当于在虚拟表上最后再加一master列, 所有值为GUOZHAO, 当然可以用数据库的变量进行运算

空进行任何运算都为空
```sql
SELECT name, salary * IFNULL(pos, 0) AS ALLsalary FROM <表名>;
```

6. 显示表数据
```sql
DESC <表名>;
```

## 条件查询

使用 where

条件筛选: AND OR XOR
