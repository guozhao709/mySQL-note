# 数据操纵语言

插入, 更新, 删除

操作的最基本单位为行

## 插入

```sql
INSERT INTO <表名> (列1, 列2, ...) VALUES (值1, 值2, ...);
INSERT INTO <表名> (列1, 列2, ...) VALUES (值1, 值2, ...),(值1, 值2, ...);
```

第一条语句为插入1个数据
第二条语句为插入多个数据

## 更新

1. 修改表中所有行的数据
```sql
UPDATE <表名>
SET 列1 = 值1, 列2 = 值2, ...
```

2. 修改指定行的数据
```sql
UPDATE <表名>
SET 列1 = 值1, 列2 = 值2, ...
WHERE 条件
```

## 删除

1. 删除表中所有行的数据
```sql
DELETE FROM <表名>
```

2. 删除指定行的数据
```sql
DELETE FROM <表名>
WHERE 条件
```
